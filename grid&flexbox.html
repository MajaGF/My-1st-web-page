CSS Grid Tutorial: Responsive Design Examples

responsive design without @media!!

grid-template-colums: repeat (auto-fit, minmax (300px, 1fr));

možemo i unutar @media screen and (min-width: 300px) ubaciti CSS od layouta

ako je jednostavniji dizajn umjesto upisivanja brojeva "grida" možemo napraviti sljedeće:

na glavnom divu = grid-template-areas: "header header" "sidebar main" "sidebar footer"

na svakom od elementata unutar diva dodati: grid-area: "naziv area"
