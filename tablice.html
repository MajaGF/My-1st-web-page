HTML tablice

HTML element <table>

HTML element <table> predstavlja dvodimenzionalnu tablicu koja se sastoji od redaka i stupaca ćelija koje sadrže podatke

<table> element -početak svake tablice (dodati klasu unutar elementa ako ćemo imati više tablica drugačijih karakteristika!)

<caption> element - naziv tablice (Prema zadanim postavkama, naslov tablice bit će središnje poravnat iznad tablice. Izgled se mijenja u CSS-u).

<thead> element - grupiranje sadržaja zaglavlja u HTML tablici.

<tr> element (table row) - definira red u HTML tablici

<th> element (table header) - definira ćeliju u HTML tablici. Dodavanjem atributa scope="col" ili scope="row" unutar ovog elementa definiramo ćeliju kao stupa ili redak.

<tbody> element - skup redaka koji sačinjavaju "tijelo" HTML tablice

<td> element - (table data cell) - definira ćeliju tablice koja sadrži podatke

<td> colspan="" element - broj stupaca koje ćemo spojiti (merge); pišemo ga u ćeliji s kojom počinjemo spajanje stupaca - OBAVEZNO prije spremanja ("save") izbrisati red u kodu koji se odnosi na stupac s kojim spajamo prvi stupac (inače neće formatirati dobro!!)

<td> rowspan="" element - broj ćelija koje ćemo spojiti (merge); pišemo ga u ćeliji s kojom počinjemo spajanje redova - OBAVEZNO prije spremanja ("save") izbrisati red u kodu koji se odnosi na red s kojim spajamo prvi prvi (inače neće formatirati dobro!!)

<tfoot> element - za prikazivanje npr. sažetka tablice


definiranje veličine/visine/širine redaka, pzadinske boje ćelije/reda/stupca, boje fonta i sl. radimo u CSS-u!!
