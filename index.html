Što je HTML?

HTML je jezik sastavljen od elemenata koji se mogu primijeniti na dijelove teksta kako bi im se dalo različito značenje u dokumentu (odlomak, paragraf, popis s grafičkim oznakama tablica ili dio tablice i sl.).

Dokument je potrebno strukturirati u logičke odjeljke (zaglavlje, stupce, navigacijski izbornik) i ugraditi sadržaje poput slika i videozapisa na stranicu.

</>

Što je CSS?

Cascading Style Sheets — ili CSS — se koristi za njegovo stiliziranje i izgled HTML dokumenta, na primjer promijenu fonta, boje, veličine i razmaka sadržaja, podijeliti sadržaj u više stupaca ili dodati animacije i druge ukrasne značajke.

HTML STRUKTURA

Svaki HTLM dokumet počinje oznakom <!DOCTYPE html>

HTML dokument počinje oznakom <html> i u osnovi je podijeljen na dva bloka HTML oznaka.

Prvi blok je zaglavlje stranice označava se oznakom <head> gdje se uvrštavaju

  • tzv. meta informacije koje nisu vidljive u prikazu web stranice (dijelovi HTML-a u stranici koje upotrebljavaju tražilice da bi zapisale informacije o vašoj stranici kao što su naslov stranice, osnovna obilježja stranice jezik stranice, kodna tablica znakova, ključne riječi, ime i prezime autora,
  • te poveznice na druge Internet stranice povezane u dokumentu

i zavšrava oznakom </head>

Drugi blok je tijelo stranice označava se oznakom <body> i sadrži sadržaj stranice prikazan u mrežnom pregledniku (tekst, slike, tablice, poveznice i sl.) završava oznakom </body>

Na kraju svakog HTML dokumenta smještene su oznake </body> i </html>


HTML komentari se ne prikazuju u pregledniku

pišu se <!-- ....... -->

CSS STRUKTURA


CSS deklaracije odvojene su točkom i zarezom, a blokovi deklaracija okruženi su vitičastim zagradama

CSS komentari se ne prikazuju u pregledniku, ali mogu pomoći u dokumentiranju izvornog koda

pišu se /* ....... */

  • CSS Selectors: .class; #id; element.class; element; * (kada želimo označiti sve elemente) element, element, element (nabrajanje više elemenata); ...
  • BACKGROUND PROPERTIES: background-color; background-image; background-repeat; background-position; backgroud; ...
  • BORDER PROPERTIES: border-style, border-width, border-color...
  • CSS OUTLINE (linija IZVAN html bordera!): outline-style; outline-color; outline-width; outline-offset ...
  • CSS FONTS: font-family; font-style; font-weight; font-variant; font-size; ...
  • CSS LINKS:
    • a:link - a normal, unvisited link
    • a:visited - a link the user has visited
    • a:hover - a link when the user mouses over it
    • a:active - a link the moment it is clicked
    • text-decoration property - mostly used to remove underlines from links
    • background-color property - used to specify a background color for links
  • CSS LISTS:
    • unordered lists (<ul>) - the list items are marked with bullets
    • ordered lists (<ol>) - the list items are marked with numbers or letters
    • list-style-type: circle | square | upper-roman | lower-alpha | none;
    • list-style-image: url(“xx.gif”);
    • list-style-position: outside (default) | inside;
  • HTML <pre> Tag - Text in a <pre> element is displayed in a fixed-width font, and the text preserves both spaces and line breaks. The text will be displayed exactly as written in the HTML source code.
  • BOX MODEL - width + border + padding + margin
  • MARGIN PROPERTIES - “prostor” izvan box-a!
  • CSS PADDING - stvaranje “mjesta” oko elemenata!
  • SHORTHAND PROPERTIES - nabrajanje bez zareza! uvijek redom: gore desno dolje lijevo
