CSS BOX MODEL

CSS Box Model

width + border + padding + margin

BORDER PROPERTIES
border-width [specific size: px, pt, cm – bez razmaka od broja!; mix: top, right, bottom, left]
border-style [dotted; dashed; solid; double; groove; ridge; inset; outset; none; hidden]
border-color [name; HEX; RGB; HSL; transparent]
border-radius – veći px/cm veća zakrivljenost okvira!
shorthand property – kad se nabrajaju properties = bez zareza! [border: 5px solid red]

CSS PADDING
stvaranje “mjesta” oko elemenata unutar box-a
padding-top, padding-right, padding-bottom, padding-left
shorthand (nabrajanje bez zareza!) = padding: xxpx xxpx xxpx xxpx
vrijednosti mogu biti: px; cm; %; inherit
ako je utvrđena vrijednost širine teksta ubaciti [box-sizing: border-box], u suprotnom se širina teskta povećava ovisno o vrijednosti zadanog paddinga!

“prostor” izvan box-a
shorthand (nabrajanje bez zareza!) = margin: xxpx xxpx xxpx xxpx
margin-top; margin-right; margin-bottom; margin-left
osim brojčanih vrijednosti može se zadati i: auto (center align) ili inherit (povezuje se na vrijednost zadane za nešto prije)
